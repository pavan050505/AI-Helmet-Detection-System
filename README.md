
# 🪖 AI Helmet Detection System  
**AI-powered real-time helmet detection with streak tracking & rewards**

An intelligent computer vision system that detects whether a bike rider is wearing a helmet using deep learning. The system also tracks helmet-wearing streaks, awards rewards, and provides a modern dashboard – all without any hardware dependency.

---

## 🚀 Project Overview

The **AI Helmet Detection System** is a full-stack AI application designed to promote road safety by detecting helmet usage in real time using a webcam or video feed.  
It encourages consistent helmet usage through **daily streak tracking**, **reward points**, and a **gamified dashboard** experience.

This project is ideal for:
- Road safety awareness
- Smart city concepts
- AI + Computer Vision portfolios
- Final year / major projects

---

## ✨ Key Features

### 🧠 Real-Time Helmet Detection
- Uses **YOLO deep learning model** for accurate helmet detection
- Works on **live webcam feed**, images, and videos
- Fast and efficient detection using OpenCV

### 🔥 Streak Tracking System
- Tracks **continuous days of helmet usage**
- Similar to **GitHub/Snapchat streak concept**
- Encourages daily safety habits

### 🎁 Rewards System
- Users earn points based on streaks
- Can be extended to coupons, badges, or achievements

### 📊 Smart Dashboard
- Shows:
  - Helmet detection status
  - Streak count
  - Reward points
  - Detection history

### 🔐 User Authentication
- Secure **login & signup**
- Personalized dashboard for each user

### 💻 100% Software Based
- No hardware required
- Works completely on **laptop/web browser**

## 🧠 How It Works

1. User opens the web dashboard
2. Starts live webcam detection
3. AI model processes each frame
4. Detects **helmet / no helmet**
5. Updates:
   - Streak count
   - Reward points
   - Detection history

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/pavan050505/AI-Helmet-Detection-System.git
cd AI-Helmet-Detection-System
```

### 2️⃣ Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```
Backend will run at:
http://localhost:5000

### 3️⃣ Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
Frontend will run at:
http://localhost:3000

## ▶️ How to Run the Project

1. Start the **backend server**  
2. Start the **frontend application**  
3. Open your browser and go to:  
   👉 **http://localhost:3000**  
4. Click **Start Detection**  
5. Allow **webcam access**  
6. The AI will start **real-time helmet detection** and automatically update **streaks & rewards**
